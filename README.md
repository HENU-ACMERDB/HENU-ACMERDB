# HENU-ACMERDB

![](./assets/image.png)

**HENU-ACMERDB** 是一个整理河南大学 ACM/ICPC 竞赛成绩的公开数据库，收录历届队伍、选手、比赛、奖项、排名和赛季大事记等信息。

## 可以了解什么

- 某位选手参加过哪些比赛，获得过哪些奖项。
- 某场比赛有哪些队伍参赛，队员是谁，最终成绩如何。
- 不同赛季的重要比赛和代表性记录。
- 选手在历史记录中的积分排名和奖牌分布。
- 不同级别比赛之间的成绩表现和排名分布。

## 如何使用

打开 [HENU-ACMERDB](https://henu-acmerdb.github.io/HENU-ACMERDB/?page=all-records) 后，可以通过导航栏查看不同页面：

- **选手查询**：输入选手姓名或拼音首字母，查看该选手的历史记录。
- **比赛查询**：选择比赛名称，查看对应比赛的队伍、成员、排名和奖项。
- **所有记录**：浏览数据库中的全部个人成绩记录。
- **选手排名**：按当前积分规则查看选手历史表现排行。
- **成绩可视化**：选择比赛级别和奖项，观察相关队伍在其它比赛中的排名分布。
- **大事记**：按赛季查看重要比赛记录。

## 数据说明

数据库主要基于队内整理的 `HENU-ACMERDB.xlsx` 维护。每次更新数据后，会生成网站使用的 JSON 数据文件并部署到 GitHub Pages。

由于早期记录可能存在缺失、格式不统一或来源不完整的情况，部分数据可能需要后续校对和补充。

## 参与补充

如果你发现记录有误，或者有缺失的比赛、队伍、选手、奖项信息，欢迎通过 Issue 或 Pull Request 补充。

提交信息时建议尽量包含：

- 比赛名称和比赛时间
- 队伍名称和队伍成员
- 排名、奖项、通过题数、罚时等成绩信息
- 数据来源或可核对的说明
